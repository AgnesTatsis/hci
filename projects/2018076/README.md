# **Εισαγωγή**

- Στόχος μου σε αυτό το μάθημα, είναι να εκμεταλλευτώ στο έπακρο την ευκαιρία που μας δίνεται να έρθουμε σε επαφή με ένα νέο λειτουργικό σύστημα άγνωστο για εμένα μέχρι τώρα. Επίσης να δημιουργήσω το προφίλ μου και να εξοικειωθώ με μία πλατφόρμα που απευθύνεται σε επιστήμονες και μαθητές της Πληροφορικής. Επιπλέον όπως και επιτυγχάνεται με την μέχρι τώρα εξέλιξη με τα βίντεο κουίζ τα οποία μας μαθαίνουν να αναζητούμε πληροφορίες από ποικίλες πηγές και μας επιτρέπουν να αναπτύξουμε ένα τρόπο σκέψης πιο "βαθύ". Εν κατακλείδι επιδιώκω να δοκιμάσω τις προσαρμοστικές μου ικανότητες σε νέα περιβάλλοντα και να βελτιώσω την διάδραση με τον ηλεκτρονικό  μου υπολογιστή, που ακόμη και σήμερα είναι πολύ φτωχή.

# **Λειτουργικό σύστημα**
### Για το Παραδοτέο02

Εγκατέστησα σε φυσικό υλικό(USB 3.0 32gb) την ανεξάρτητη διανομή Linux το ArchLinux! Περιληπτικά πρόσθεσα 
- ένα window manager, συγκεκριμένα το i3,
- μια μηχανή αναζήτησης, 
- το nautilus file manager

κι έπειτα από κάποιες τροποποιήσεις είναι πλέον πλήρως λειτουργικό, ειδικά διαμορφωμένο και προσαρμοσμένο έτσι, ώστε να ικανοποιεί τις ανάγκες μου!
