# Παραδοτέο 1: Εισαγωγή
Με το μάθημα "Επικοινωνία Ανθρώπου-Υπολογιστή", επιδιώκω να μάθω πως λειτουργούν εναλλακτικά λογισμικά, όπως το archlinux και να γνωρίσω συνάμα τις δυνατότητές μου σε τέτοιο περιβάλλον. Ως προσωπικό έργο, θα ήθελα να αναπτύξω παιχνίδι μορφωτικκών γνώσεων όπου θα έχει άμεση αλληλεπίδραση ο χρήστης με το σύστημα. Η ιδέα προήλθε από μαθήματα της σχολής στα οποία κατάφερα να δημιούργησω δύο παιχνίδια, ένα γνώσεων και ένα δράσης.
