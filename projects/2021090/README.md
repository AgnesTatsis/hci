## Εβδομάδα Δεύτερη - Άσκηση γραμμής εντολών (linux install)

<br /> 

H εγκατάσταση του arch linux ήταν μια πρόκληση στην αρχή. Έπειτα από δοκιμές σε virtual box, μερικές αποτυχιμένες προσπάθειες σε πραγματικό υλικό και πονοκεφάλους, 
επιτέλους η εγκατάσταση ήταν επιτυχής! Ένα live usb και η βοήθεια του arch wiki και άλλων πηγών κυρίως tutorial στο youtube για την αντιμετόπηση των προβλημάτων(όπως η απενεργοποίηση του secure boot!) ήταν αρκετά για να επιστρέψει το παλίο λάπτοπ μου στην ενεργό δράση.
Η διαδικκασία που ακολούθησα για την εγξατάσταση ήταν η εξής:
Πρώτο βήμα ήταν η δοκιμή σε εικονική μηχανή ,συγκεκριμένα το **Virtual Box**. Για έναν αρχάριο σαν εμένα οι οδηγίες του [arch wiki](link),αν και ήταν μεν σαφής εξακολουθούσαν να είναι *απειλητικές*.Δοκίμασα λοιπόν ένα νέο tutorial στο youtube με έναν ευκολότερο τρόπο με ένα script από την βιβλιοθήκη archinstall. Με αυτό τον τρόπο τα βασικότερα σημεία της εγκατάστασης εμφανίζονται σε μορφή πίνακα όπου καλήσε να επιλέξεις τις ρυθμήσεις σου και στο τέλος εκτελείται το σκριπτ εγκατάστασεις. Ο τρόπος αυτός ήταν ο  ιδανικός ώστε να αποκτήσω μια πρώτη εικόνα για τις παραμέτρους που πρέπει να λάβω υπόψη και να προσδιορίσω ,όπως τα partition,το format,τα directories.Παρόλα αυτά για τον απόλυτο έλεγχο η εγκατάσταση πρέπει να γίνει με το χέρι.Έτσι ακολουθώντας ένα ένα βίντεο tutorial,το οποίο έγινε ο βασικός οδηγός μου στην εγκατάσταση σε πραγματικό υλικό. Ακολουθώντας αυτό πλέον είχα μια διανομή arch linux στην εικονική μηχανή του υπολογιστή μου.
Πλέον ήταν η ώρα για εγκατάσταση σε πραγματικό υλικό.Το παλιό μου lenovo λάπτοπ ήταν ο ιδανικός υποψήφιος-θύμα. Με windows 8,πάμπολα antivarius να τρέχουν στο backround,και μπαταρία που συνεχώς ζητάει τροφοδοσία,ο υπολογιστής σέρνονταν.Έπρεπε να γίνει ολική διαγραφή του προήγουμενου λογισμικού πράγμα που διευκόλυνε κατά πολύ την διαδικασία καθώς και λάθος να συμβεί στα partition δεν θα επιρεαστούν τα συμαντικά δεδομένα μου.Ο διαχωρισμός σε partitions ίσως και να ήταν το πιο βαρετό κομμάτι της διαδικασίας.fdisk και δημιουργία νέων partitions,ένα για το EFI και για τα linux. To home direktory δεν είχε αυτή τη φορά ένα κουμπάκι που απλά πατας *yes*.Πρέπει να φτιαχτεί από την αρχή ,mkdir home, και να γίνει mount στα σωστά volumes με την εντολή mnt στο αντίστιχο σημείο.Με τους δίσκους να έχουν δημιουργηθεί πλέον σειρά έχουν *the fun stuff*.Εγκατάσταση πακέτων με την pacman -S  όνομα πακέτου (σημείωση:για να εκτελεστεί αυτή η εντολή πρέπει να είσαι root).Τα πολύ βασικά ήταν τα linux headers,nano για επεξεργασία κειμένου για τα configurations,networkmanager για σύνδεση στο ίντρνετ αργότερα χωρίς το καώδιο του ethernet.Τελειώνοτνας με αυτά για αυτοματοποιήμένο boot κατέβασα grub ως bootloader όπως το tutorial.Υπήρχε ένα τελευταίο εμπόδιο.Οι ρυθμήσεις UEFI δεν έιχαν αλλάξει!Έπρεπε να γίνει από την αρχή επιστροφή στα deafault και απενεργοποίηση του secure boot για την επιτυχή εγκατάσταση!!!Ξανά η ίδια διαδικασία και πλέον ένα archlinux με τερματικό ήταν έτοιμο.Με την βοήθεια τουtutorial κατέβασα αργότερα το xmonad window manager ώστε να μην δουλεύω σε εντελώς γυμνό περιββάλλον.
