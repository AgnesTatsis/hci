# Παραδοτέο 1: Εισαγωγή
Μέσα από το μάθημα "Επικοινωνία Ανθρώπου-Υπολογιστή", επιδιώκω να εμπλουτίσω τις γνώσεις μου σχετικά με τον τρόπο που αναπτύσσονται οι σχέσεις και οι ενέργειες ανάμεσα στον χρήστη και το μηχάνημα. Επιθυμώ να κατανοήσω την έννοια της "ψηφιακής" ομάδας και της συνεργατικότητας, καθώς εντάσσομαι ήδη σε μια "φυσική" ομάδα (Σώμα Ελληνικού Οδηγισμού) και με όλα τα προβλήματα που θα προκύψουν από την επικοινωνία με τα "ψηφιακά" πρόσωπα, να μπορέσω να βελτιώσω τις συνεργατικές μου σχέσεις σε γενικό επίπεδο.
