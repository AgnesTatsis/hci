# Παραδοτεο 4ο
## Warm-Up
### Sampler
Το Sampler είναι ένα πρόγραμμα terminal το οποιο εχει πολλες δυνατοτητες παρακολούθησης του συστήματος του χρηστη αλλα και server. Εγω το χρησιμοποιήσω για να παρακολουθω τον επεξεργαστη μου και τον χρονο.

[![asciicast](https://asciinema.org/a/YSqF5OyDI7Z7tgG4ro556EUPH.svg)](https://asciinema.org/a/YSqF5OyDI7Z7tgG4ro556EUPH)

### Googler
Το Googler ειναι ενα implementation του google search σε terminal με πιο εξειδικευμενους τρόπους αναζήτησης πχ αναζητηση σε ενα website και συγκεκριμενου filetype. Εγω εκανα μια αναζητηση απλη και μια ακόμα, με target το skroutz.gr.

[![asciicast](https://asciinema.org/a/8c66eo1dSP4Idw4rOKexTY360.svg)](https://asciinema.org/a/8c66eo1dSP4Idw4rOKexTY360)
